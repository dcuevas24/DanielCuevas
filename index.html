Portafolio (Proyectos Destacados)

Bobinando

Canal de YouTube

Fecha: 21/01/2018

Puedes verlo en: https://www.youtube.com/bobinando

Bobinando es un proyecto en el cual se enseñan materias de ingeniería tales como circuitos eléctricos, termodinámica, accionamientos eléctricos entre otras materias. En este canal la ingeniería se explica de forma fácil, rápida y sencilla.

Sitio web de Bobinando

WordPress

Fecha: 27/04/2020

Puedes verlo en: https://www.bobinando.com/

El sitio web de Bobinando fue el resultado de 4 meses de trabajo en WordPress para crear una plataforma donde se puede estudiar, repasar y leer sobre ingeniería. Cada semana se encuentra nuevo contenido.
